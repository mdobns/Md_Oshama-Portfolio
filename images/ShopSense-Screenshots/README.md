## How to Capture Screenshots

1. Run the development server: `python manage.py runserver`
2. Navigate to http://127.0.0.1:8000/
3. Use your browser's screenshot tool or:
   - **Windows**: Windows + Shift + S
   - **macOS**: Command + Shift + 4
   - **Linux**: Print Screen or Shift + Print Screen
4. Crop and save as PNG format
5. Place in this directory with the exact filenames listed above

## Screenshot Guidelines

- **Resolution**: At least 1920x1080 for desktop views
- **Format**: PNG with transparency where applicable
- **Quality**: High quality, no compression artifacts
- **Content**: Show real product data, not lorem ipsum
- **Privacy**: Remove any personal/sensitive information
- **Annotations**: Optional - add arrows or highlights for key features
